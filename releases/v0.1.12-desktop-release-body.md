## What's included

- HAC / HACD / BTC send & receive
- L2 Fast Pay channel management
- Quantum (Type 4) transactions with **dynamic fee** from node (~0.004 HAC, not fixed 40 HAC)
- Air-gap signing for quantum sends
- WebAuthn / Windows Hello unlock
- Dust Whisper messenger (optional local relay)

## Requirements

- Windows 10/11 x64
- Internet access to `nodeapi.hacash.org`

## Install

1. Download `hacash-wallet-desktop-v0.1.12-x64-setup.exe`
2. Run the installer and follow prompts
3. Open **Hacash Wallet** from Start menu
