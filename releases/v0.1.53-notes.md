# Hacash Wallet v0.1.53

## Fixes
- **USD prices:** primary source is CoinPaprika (CoinGecko free tier often returns HTTP 429). Quotes work again on mobile + desktop.
- **Home UI:** shows a clear “USD quotes offline” note when prices cannot load (balances still work).
- **In-app updater:** accepts SHA-256 from GitHub digests **or** `SHA256SUMS-*.txt`; desktop prefers **MSI**; always offers “Open release page”.
- **Desktop install:** use **MSI** or **portable .exe** if setup.exe is blocked by SmartScreen / fails to run.

## Install tips (Windows)
1. Prefer `hacash-wallet-desktop-v0.1.53-x64.msi`
2. Or run portable `.exe` (no install)
3. If SmartScreen appears: More info → Run anyway (only from the official GitHub release)
