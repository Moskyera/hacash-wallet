## Hacash Wallet Mobile v0.1.4

Fixes Android release loading `127.0.0.1:1421` dev server instead of bundled UI.

### Root cause
Mobile `Cargo.toml` was missing the `custom-protocol` feature (desktop had it). Without it, Tauri `is_dev()` is always true on Android.

### Fixes
- Enable `custom-protocol` in mobile Rust build (default feature)
- Strip `devUrl` from bundled `tauri.conf.json` in APK assets
- Bundle Vite `dist/` into APK `assets/`

### Install
Download `hacash-wallet-mobile-v0.1.4-arm64.apk` (arm64, signed release).
