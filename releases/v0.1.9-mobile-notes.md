# Hacash Wallet Mobile v0.1.9

## Download

- **APK (arm64):** `hacash-wallet-mobile-v0.1.9-arm64.apk`

## Changes

### Fast Pay UI

- Shorter, clearer Fast Pay status messages (no dashes, less jargon)
- **Enable** / **Disable** buttons placed below the status on Home and More → Fast Pay
- **Enable** opens the payment channel (~10 HAC deposit)
- **Disable** closes the channel and returns funds on-chain

## Install

1. Download the APK
2. Allow install from unknown sources if prompted
3. Open the app and unlock your wallet
