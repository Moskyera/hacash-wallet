## Hacash Wallet Mobile v0.1.25

**Critical fix:** v0.1.22–v0.1.24 could crash immediately on launch (duplicate FileProvider in Android manifest).

### Changes
- Fix startup crash: single FileProvider entry in AndroidManifest
- More reliable in-app APK install (direct installer intent + URI permissions)

### If the app closes as soon as you open it
1. Uninstall Hacash Wallet (wallet data stays in app storage until uninstall — export backup first if needed)
2. Install this APK manually from the browser
3. Re-import wallet from your backup if you uninstalled

### If you are on v0.1.21 or older
Install this APK once manually, then future updates via **Settings → App update** should work.

### Permissions
If prompted, enable **Install unknown apps** for Hacash Wallet, then tap **Download & install** again.
