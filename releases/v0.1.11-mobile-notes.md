# Hacash Wallet Mobile v0.1.11

## Download

- **APK (arm64):** `hacash-wallet-mobile-v0.1.11-arm64.apk`

## Changes

### Biometric unlock

- **More → Security → Biometric unlock** — enable with passphrase + fingerprint
- **Unlock screen** — **Unlock with biometric** button when enabled
- Separate from **Biometric confirm** (large sends)

### Private key (advanced)

- **More → Security → Private key** — reveal after passphrase, auto-hide in 60s

### Receive copy

- **HACD** — shorter, clearer text
- **BTC** — wallet receive address (not Fast Pay)

### UI fix

- Header (logo, wallet name, address) respects status bar safe area on Android

### Also includes

- Passkey fix, biometric confirm, hub auto-discovery (v0.1.10 chain)
