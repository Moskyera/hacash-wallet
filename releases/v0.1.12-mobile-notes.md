# Hacash Wallet Mobile v0.1.12

## Download

- **APK (arm64):** `hacash-wallet-mobile-v0.1.12-arm64.apk`

## Changes

### Type 4 quantum fee fix (important)

- **Before:** fixed `40:244` fee (~40 HAC) — wrong, blocked normal sends
- **Now:** dynamic fee from node `/query/fee/average` (~**0.004 HAC** for ~5 KB Type 4 tx)
- Preflight shows: balance · fee · total needed

Example: send 100 HAC now needs ~**100.004 HAC** on your hybrid address, not 140 HAC.
