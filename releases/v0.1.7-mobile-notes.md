# Hacash Wallet Mobile v0.1.7

## Fixes

- **Delete wallet** — More → Security → Delete wallet from device (needed before creating a truly new wallet)
- **Test node** button in Network settings
- Clearer node errors (includes GrapheneOS Network permission hint)
- Unlock screen shows your wallet address
- Node ping uses a working API endpoint

## GrapheneOS network (important)

If you see `cannot reach http://nodeapi.hacash.org`:

**Settings → Apps → Hacash Wallet → Permissions → Network → Allow**

Then in the app: **More → Network → Test node**

## Why "new wallet" showed the same address

Android keeps wallet data when you update the APK. "Create wallet" only works when no wallet exists on the device. Use **Delete wallet** first, or **Settings → Apps → Hacash Wallet → Storage → Clear storage**.
