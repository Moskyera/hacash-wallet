## Hacash Wallet Mobile v0.1.26

**Must install over v0.1.24 crash:** older broken builds used the same Android `versionCode` (1000), so the phone may have kept the broken APK even after you tapped install.

### Fixes
- **versionCode 1026** — Android will actually replace v0.1.22–v0.1.25 installs
- Startup no longer aborts if wallet vault cache warm fails
- Single FileProvider in manifest (v0.1.25 fix retained)
- ProGuard keep rules for APK installer

### Install steps (important)
1. **Uninstall** Hacash Wallet (Settings → Apps). Export backup first if the app still opens.
2. Download this APK and install.
3. In app: **More → Settings → App update** should show **v0.1.26**.

If it still closes, tell us your phone model and Android version.
