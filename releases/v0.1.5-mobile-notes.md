## Hacash Wallet Mobile v0.1.5

Fixes Android HTTP access to the Hacash public node API (`http://nodeapi.hacash.org`).

### Root cause
Android 9+ blocks cleartext HTTP by default. The Hacash node API has no TLS. Release builds also set `usesCleartextTraffic=false`, and the prior `network_security_config` only listed domain exceptions without a permissive `base-config` (needed for Rust `reqwest` as well as WebView).

### Fixes
- `android-network-security.xml`: `<base-config cleartextTrafficPermitted="true">`
- `apply-android-patches.ps1`: sets `manifestPlaceholders["usesCleartextTraffic"] = "true"` for release
- `useWalletSession.ts`: `loadWalletData` uses `Promise.allSettled` for graceful partial failures

### Install
Download `hacash-wallet-mobile-v0.1.5-arm64.apk` (arm64, signed release).
