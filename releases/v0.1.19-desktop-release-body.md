# Hacash Wallet Desktop v0.1.19

## What's new

- **New logo** (`dnhw.png`) — high-quality glossy brand mark
- **HACD Connect** on Home — connect wallet to hacd.it from browser (no embedded launchpad)
- **Browser bridge** at `http://127.0.0.1:9477` + Chrome/Edge extension (`apps/desktop/hacd-browser-bridge`)
- **Pause auto-lock on HACD** — Privacy setting (default on)
- Heartbeat from browser extension prevents lock while hacd.it tab is open
