# Hacash Wallet Desktop v0.1.18

## What's new

- **Dynamic L1 fees** for HAC, HACD, and BTC on-chain sends
- Fee estimated from node mempool purity × transaction wire size (+10% headroom)
- Fast Pay (L2) hub fee still comes from the payment hub
