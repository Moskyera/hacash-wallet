# Hacash Wallet Mobile v0.1.16

## Download

- **APK (arm64):** `hacash-wallet-mobile-v0.1.16-arm64.apk`

## What's new

- **Home:** portfolio total in USD (HAC + BTC + HACD estimate)
- **Home:** wallet version badge (v0.1.16)
- **Home:** redesigned HAC / HACD / BTC balance cards (3-column grid)

## Install (Android / GrapheneOS)

1. Download APK in browser
2. Files → Downloads → tap APK
3. Allow install if prompted
4. Open **Hacash Wallet**
