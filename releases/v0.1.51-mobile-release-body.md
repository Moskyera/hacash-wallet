# Hacash Wallet Mobile v0.1.51

Android arm64 APK (versionCode 151).

## Changes
- Bottom navigation: **HACD** tab (replaces Chat on the main bar; Chat stays under More)
- Network settings: **official node API** (`http://nodeapi.hacash.org`) is the default; custom node is opt-in
- Language switcher moved from the home overlay into **More → Language**
- Version display and MoneyNex inject use **v0.1.51**
- Includes v0.1.50 fixes: Android IPC crash fix, Explorer-style HACD metadata cards

## Install
1. Download `hacash-wallet-mobile-v0.1.51-arm64.apk`
2. Allow install from the browser/Files if prompted
3. Open **Hacash Wallet**

SHA-256 is in `SHA256SUMS-v0.1.51-mobile.txt`.
