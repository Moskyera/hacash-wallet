# Hacash Wallet Mobile v0.1.24

## Download

- **APK (arm64):** `hacash-wallet-mobile-v0.1.24-arm64.apk`

## What's new

- **Fix in-app updater** — install prompt no longer silently fails; clearer errors
- **Open in browser** fallback if in-app install does not start
- Prompts for **Install unknown apps** permission when needed
- `mosky.png` launcher icon + header mark (from v0.1.23)

## Install note

If you are on v0.1.21 or v0.1.22 and in-app update failed, **download this APK manually** once. Later updates work in-app.
