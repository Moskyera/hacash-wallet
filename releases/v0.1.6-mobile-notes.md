# Hacash Wallet Mobile v0.1.6

Fixes **node API connection errors** on unlock when the saved node URL was wrong (e.g. `http://nodeapi.org`).

## Changes

- Auto-correct `nodeapi.org` → `http://nodeapi.hacash.org` on settings load and save
- Force HTTP (not HTTPS) for the official Hacash node API
- Android: use blocking HTTP client for node requests (more reliable on GrapheneOS)
- Longer connect/read timeouts for slow mobile networks

## If you still see errors

1. **More → Settings → Network** — set node URL to exactly: `http://nodeapi.hacash.org`
2. Uninstall the old APK, then install this build
3. Ensure the phone has internet (Wi‑Fi or mobile data)

## Install (GrapheneOS / Android)

1. Download `hacash-wallet-mobile-v0.1.6-arm64.apk` from GitHub Releases
2. Open with Vanadium or Files → Install
3. Allow “Install unknown apps” if prompted
