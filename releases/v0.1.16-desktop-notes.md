# Hacash Wallet Desktop v0.1.16

Desktop release with Fast Pay hub discovery, private key reveal, YubiKey setup guide, matching launcher icon, and quantum fork design docs.

## Downloads

### Windows x64

| File | Description |
|------|-------------|
| `hacash-wallet-desktop-v0.1.16-x64-setup.exe` | **Recommended** — NSIS installer |
| `hacash-wallet-desktop-v0.1.16-x64.msi` | MSI installer (enterprise) |
| `hacash-wallet-desktop-v0.1.16-x64-portable.exe` | Portable executable |

### Linux x64

| File | Description |
|------|-------------|
| `hacash-wallet-desktop-v0.1.16-x64.AppImage` | **Recommended** — runs on most distros |
| `hacash-wallet-desktop-v0.1.16-x64.deb` | Debian / Ubuntu package |
| `hacash-wallet-desktop-v0.1.16-x64-portable` | Standalone binary (no install) |

## What's new

- **Fast Pay:** hub auto-discovery — scan online providers and apply with one click
- **Settings:** reveal private key (passphrase required, auto-hides in 60s)
- **Security:** step-by-step YubiKey / Windows Hello setup instructions
- **Launcher icon:** desktop app icon matches mobile (gold mark on black)
- **Docs:** community handoff guide + Hacash Quantum Wallet design (fork plan for later)

## Also included (unchanged)

- True black UI theme, HAC / HACD / BTC send & receive
- L2 Fast Pay channel management
- Quantum (Type 4) with dynamic fee (~0.004 HAC)
- Air-gap signing, WebAuthn / Windows Hello, Dust Whisper

## Requirements

- Windows 10/11 x64 or Linux x64 (glibc)
- Internet access to `nodeapi.hacash.org` (or your own node)

## Install (Windows)

1. Download `hacash-wallet-desktop-v0.1.16-x64-setup.exe`
2. Run the installer (SmartScreen may warn — binary is unsigned)
3. Open **Hacash Wallet** from Start menu

## Install (Linux)

**AppImage:** `chmod +x hacash-wallet-desktop-v0.1.16-x64.AppImage && ./hacash-wallet-desktop-v0.1.16-x64.AppImage`

**Debian/Ubuntu:** `sudo dpkg -i hacash-wallet-desktop-v0.1.16-x64.deb`
