# Hacash Wallet Mobile v0.1.8

Fixes **Quantum → Node unreachable** when using the public Hacash node (`http://nodeapi.hacash.org`).

## Cause

The node health check called endpoints that fail on the public API (`/query/metrics` = 404, balance probe = invalid JSON).

## Fix

Node ping / Quantum health now uses `GET /query/latest` (block height), which works on the public node.

## Quantum quick start

1. **More → Network** — node URL: `http://nodeapi.hacash.org`
2. **More → Quantum** — enable Quantum mode
3. Create **PQC** or **Hybrid** account (keystore password)
4. Fund quantum address: send HAC from main wallet to quantum address (legacy Type 1)
5. **Node health** should show `Node reachable · height …`
6. Type 4 send from Quantum tab
