# Hacash Wallet Desktop v0.1.51

Windows x64 installers (setup + MSI + portable). Linux packages are attached by CI when the tag is pushed.

## Changes
- Network settings: **official node API** is the default; custom node is opt-in
- Language switcher in **Settings** (removed fixed home overlay)
- HACD gallery and metadata cards (Explorer style)
- Version **v0.1.51**

## Windows
- **Setup (recommended):** `hacash-wallet-desktop-v0.1.51-x64-setup.exe`
- MSI: `hacash-wallet-desktop-v0.1.51-x64.msi`
- Portable: `hacash-wallet-desktop-v0.1.51-x64-portable.exe`

## Linux (CI)
- AppImage, `.deb`, portable binary (uploaded after GitHub Actions finishes)
