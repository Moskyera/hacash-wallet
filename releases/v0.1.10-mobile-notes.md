# Hacash Wallet Mobile v0.1.10

## Download

- **APK (arm64):** `hacash-wallet-mobile-v0.1.10-arm64.apk`

## Changes

### Passkey (WebAuthn) on Android

- Uses the real app origin (`https://tauri.localhost`) instead of desktop `localhost:1420`
- Register passkey uses device platform authenticator (fingerprint/face)

### Biometric confirm

- **More → Security → Biometric confirm**
- Toggle **Use biometric for sends**
- **Test biometric** button to verify fingerprint works
- Used automatically for sends ≥ 100 HAC (Balanced profile)

### Also includes

- Hub auto-discovery (v0.1.10 chain)
- Fast Pay UI improvements from v0.1.9
